# BUILD with CANDIDE
CANDIDE wallet is designed as open-source. It is focused on building on standards, compatibility and interoperability with Ethereum. With CANDIDE, you can integrate your dapp with the wallet, help contribute to the codebase or fork a difference interface. The playground is yours! 

If you have any questions, feel free to join our [Discord chat](https://discord.gg/Q8B6WJ2MvT)
